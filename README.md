# Fitness CRM Tests

Автотесты для проверки CRM системы тренера.

## Что покрыто:

- UI тесты (Selenium)
    - логин (успешный / неуспешный)
    - создание клиента
    - открытие клиента

- API тесты
    - авторизация
    - работа с клиентами

## Стек:

- Java
- Selenium
- JUnit 5
- Gradle

## Как запускать:

```bash
./gradlew test
